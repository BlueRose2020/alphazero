
# 配置模块说明

本目录用于管理项目的各种配置和类型定义，确保代码结构清晰、易于维护和扩展。

## 文件列表与功能简介

- **basic.py**：基础配置项，包含项目通用参数，alphzero基本参数等
- **quick_model_config.py**：快速模型配置，支持根据配置文件自动调整模型结构，便于快速实验和验证。
- **train_config.py**：训练相关参数配置，包含训练流程、超参数，多进程等。
- **type_alias.py**：类型别名定义，提升代码可读性和类型安全。
- **ui.py**：UI 相关配置，定义界面参数和样式。
- **utils_config.py**：工具类配置，辅助功能参数，包括经验池，数据增强，日志等
